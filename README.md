## 这是什么？

这是一个用于《恋与深空》的简易芯核筛选工具，旨在帮助玩家快速判断芯核的价值。

目前主要用于回答：

-   这颗芯核能不能留？
-   适合哪个当前使用身份和流派？
-   主副属性是否有效？
-   **词条量**是否足够，**集中度**如何？
-   **数值质量**是传家宝/优秀/普通/偏低？
-   能不能赌？好不好赌？怎么赌？

当前版本仍为测试版，结果仅供参考，欢迎捉虫和提供建议。

## 核心判断原则

本工具严格遵守以下原则，以提供更准确的参考：

### 1. 不换算原则
绝不将固定数值（小攻击/小生命/小防御）换算成百分比加成。固定数值和百分比加成是两种不同的词条，后期收益差距明显，本工具会分开判断它们的价值。

### 2. 词条量与集中度
根据攻略组思路，芯核的价值不仅看词条类型，也看“词条量”是否足够，以及是否集中在有效词条上。
-   **总词条量** = `(可见副属性数量 × 2) + (所有强化次数总和)`
-   **有效词条量** = 核心/优秀/可用副词条的基础词条量 + 强化在它们上面的次数
-   **集中度** = `(有效词条量 / 总词条量) * 100%`

### 3. 数值质量
对于百分比加成类词条，在不同强化等级下，数值越高，质量越好。本工具会根据攻略数据，给出“传家宝/优秀/普通/偏低”的参考。

## 当前支持

-   **男主**：夏以昼
-   **身份**：冥罗之主 / 终极兵器 X-02 / 远空执舰官
-   **芯核位置**：α / β / γ / δ (自动识别日/月卡)
-   **阶数**：日卡(双0~双3) / 月卡(0~3)
-   **流派**：通用 / 虚弱流 / 双暴流
-   **玩家阶段**：新手 / 后期
-   **赌核建议**：根据强化等级和词条命中情况，给出“怎么赌”的建议。

---

## 使用方式

1.  选择**当前使用身份**、**芯核位置**。
2.  工具会自动识别为日卡/月卡，并切换对应的**阶数**选项。
3.  根据需求选择**玩家阶段**、**流派**等战斗配置。
4.  选择**主属性**、**强化等级**。
5.  勾选**副属性**，并填写**强化次数**和**当前数值**（数值为选填）。
6.  点击「开始判断」。
