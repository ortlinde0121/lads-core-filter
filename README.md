## 🌟 V1.1 秦彻觉醒版 重磅更新！

经过底层逻辑的全面重构，《芯核能好怎》正式迈入 V1.1 阶段！本次更新带来了海量新内容：

- **🦇 秦彻全面实装**：新增秦彻三大身份的精准判断逻辑。
  - **银翼恶魔（蓝弧）**：防御特化，精准区分虚弱流与双暴流。
  - **深渊主宰（绿珥）**：生命特化，血牛狂战士专属逻辑。
  - **无尽掠夺者（粉珀）**：攻击特化，纯双暴刺客判定。
- **🎨 多男主动态主题**：新增“变色龙”框架。选择不同男主时，网页主题色、背景渐变、提示框将自动切换为该男主的官方应援色（如秦彻的红黑恶霸色）。
- **🛡️ 净网术语替换**：为适应各大社区平台的审核机制，将工具内所有的“赌核”相关术语替换为更安全、更符合游戏语境的“强化/升”（如：能升吗、好升吗、怎么升）。

---

## 🛠️ 核心功能保留

- **AI 传图自动打勾 (Beta)**：纯前端 OCR 技术，上传截图自动识别副属性。
- **全属性动态数值评价**：根据强化次数，动态计算生命、攻击、防御及各类加成的数值质量（传家宝/优秀/普通/偏低）。
- **新手/后期双模式**：动态调整固定数值（小攻击/生命/防御）的评分权重。
