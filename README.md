# lads-core-filter
# 简易芯核筛选器 V0.6

by 阿泽

这是一个用于《恋与深空》的简易芯核筛选小工具。

## 当前支持

当前主要支持夏以昼：

- 远空执舰官（红漪）
- 冥罗之主 / 鬼王（绿珥）
- 终极兵器 X-02 / 哥达（紫辉）

## 功能

- 根据身份、芯核位置、阶数、谱向、流派判断芯核适配度
- 支持 α / β / γ / δ 官方位置
- 支持副属性强化次数填写
- 输出：
  - 能不能留
  - 能不能赌
  - 好不好赌
  - 怎么赌
  - 一句话结论版
  - 评分细则版

## 使用方式

打开网页后按顺序选择：

1. 男主与身份
2. 芯核位置
3. 阶数、谱向、流派
4. 主属性
5. 副属性与强化次数
6. 点击「开始判断」

## 注意事项

本工具仅供参考，不是绝对标准。  
芯核判断会受到练度、队伍、操作手法、关卡环境等影响。
