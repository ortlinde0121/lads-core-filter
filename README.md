## 🌟 V1.0 正式版更新

《芯核能好怎》V1.0 正式版上线啦！

本工具从最初的手动芯核判断器，升级为带有 **截图识别辅助 + 全属性动态数值评价** 的半自动芯核筛选工具。

### V1.0 主要功能

#### 1. 📸 传图自动勾选副属性 Beta

可以上传芯核属性截图，工具会尝试自动识别副属性名称，并帮你勾选对应词条。

目前支持识别：

- 生命
- 生命加成
- 攻击
- 攻击加成
- 防御
- 防御加成
- 暴击
- 暴伤
- 誓约增伤
- 虚弱增伤

> 注意：截图识别是辅助功能，不是强制功能。  
> 如果识别不准，也可以完全手动勾选和修改。

建议上传时尽量裁剪，只保留芯核属性文字区域，识别会更稳定。

---

#### 2. 📊 全属性动态数值评价

V1.0 已支持对所有副属性进行数值质量判断。

包括：

- 生命
- 攻击
- 防御
- 生命加成
- 攻击加成
- 防御加成
- 暴击
- 暴伤
- 誓约增伤
- 虚弱增伤

工具会根据该词条的 **强化次数**，动态计算当前数值在理论区间中的位置，并给出：

- 传家宝
- 优秀
- 普通
- 偏低

的评价。

> 例如：同样是攻击加成，不会简单按芯核总等级判断，而是根据“攻击加成被强化了几次”来判断数值质量。

---

#### 3. 🎲 赌核建议

工具会结合以下信息，给出“能不能赌 / 好不好赌 / 怎么赌”的建议：

- 当前身份
- 芯核位置
- 主属性
- 副属性
- 强化次数
- 当前强化等级
- 玩家阶段
- 流派选择

输出包括：

- 建议上锁
- 好赌
- 小赌
- 谨慎赌
- 暂停观察
- 停手

---

#### 4. 🌱 新手 / 后期双模式

考虑到不同阶段玩家的养成差异，工具加入了玩家阶段判断。

- 新手模式：适合四星卡为主、资源有限、需要过渡芯核的玩家  
- 后期模式：适合五星卡为主、追求毕业和高质量芯核的玩家  

新手模式下，固定数值类词条如生命、攻击、防御会获得更高权重，避免误融前期好用的过渡芯核。

---

#### 5. 💙 夏以昼身份支持

当前主要支持夏以昼相关身份：

- 冥罗之主 / 鬼王
- 终极兵器 X-02 / 哥达
- 远空执舰官

并区分：

- 虚弱流
- 双暴流
- 通用判断

不同身份、不同流派下，词条优先级会动态变化。

## 📝 使用方法

### 方法一：手动输入

1. 选择男主和身份
2. 选择芯核位置
3. 选择玩家阶段、谱向、流派
4. 选择主属性和强化等级
5. 勾选副属性
6. 填写副属性强化次数
7. 可选填具体数值
8. 点击「开始判断」

---

### 方法二：截图辅助识别

1. 点击「上传芯核截图」
2. 上传裁剪后的芯核属性截图
3. 等待工具自动识别并勾选副属性
4. 人工核对识别结果
5. 手动填写强化次数和数值
6. 点击「开始判断」

> OCR 识别只负责辅助勾选副属性，强化次数和具体数值仍建议玩家手动填写，以保证判断准确。

## ⚠️ 注意事项

- 本工具仅作为芯核筛选和赌核辅助，不代表唯一标准。
- 实际养成仍需结合自身卡池、练度、队伍、关卡谱向和操作习惯。
- OCR 识别功能为 Beta，可能存在识别不准、加载较慢等情况。
- 第一次使用截图识别时，浏览器可能需要加载中文识别模型，请耐心等待。
- 星谱颜色暂不参与评分，请玩家根据关卡需求自行凑色破盾。
