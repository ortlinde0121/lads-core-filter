# 简易芯核筛选器

> 《恋与深空》芯核初筛 / 赌核参考小工具  
> by 阿泽

在线使用：  
https://ortlinde0121.github.io/lads-core-filter/

---

## 这是什么？

这是一个用于《恋与深空》的简易芯核筛选工具。

目前主要用于判断：

- 这颗芯核能不能留
- 适合哪个当前使用身份
- 主属性是否合适
- 副属性是否有效
- 强化次数有没有打到核心词条
- 能不能赌
- 好不好赌
- 怎么赌、赌到几级该停

当前版本仍为测试版，结果仅供参考。

---

## 当前支持

当前主要支持：**夏以昼**

已支持当前使用身份：

- 远空执舰官
- 冥罗之主 / 鬼王
- 终极兵器 X-02 / 哥达

> 注意：  
> 本工具按“当前使用身份”判断芯核收益，不按卡片来源判断。

例如：  
即使你的日卡来自某一套卡，只要实战中选择的是冥罗之主身份，工具就会按冥罗之主的属性需求进行判断。

---

## 已支持芯核位置

工具已按官方芯核位置区分日卡 / 月卡：

| 位置 | 类型 | 说明 |
|---|---|---|
| α | 日卡芯核 | 日卡左芯核，主属性固定生命 |
| β | 日卡芯核 | 日卡右芯核，方块 / 功能位 |
| γ | 月卡芯核 | 月卡左芯核，主属性固定攻击 |
| δ | 月卡芯核 | 月卡右芯核，三角位 |

---

## 已支持功能

- 根据当前使用身份判断属性收益
- 根据 α / β / γ / δ 自动切换主属性池
- 自动识别日卡芯核 / 月卡芯核
- 日卡使用“双0阶 / 双1阶 / 双2阶 / 双3阶”
- 月卡使用“0阶 / 1阶 / 2阶 / 3阶”
- 支持选择谱向、流派、鬼王打法
- 支持勾选副属性
- 支持填写副属性强化次数
- 输出基础评级
- 输出赌核评级
- 输出：
  - 能赌吗
  - 好赌吗
  - 怎么赌
- 生成两种复制文案：
  - 一句话结论版
  - 评分细则版

## 使用方式

打开网页后，按顺序填写：

1. 选择当前使用身份
2. 选择芯核位置 α / β / γ / δ
3. 选择阶数、谱向、流派
4. 选择主属性
5. 勾选副属性
6. 如果芯核已经强化过，填写对应副属性右侧的强化次数
7. 点击「开始判断」

结果区会直接显示：

- 基础评分
- 基础评级
- 赌核评级
- 能不能赌
- 好不好赌
- 怎么赌

## 关于星谱颜色

从 V0.7 开始，星谱颜色不参与评分。

实战中配队通常为：

- 2 张日卡
- 4 张月卡

不同颜色芯核都有可能根据关卡、配队和破盾需求参与搭配。

因此本工具目前只判断芯核属性是否适合当前使用身份，不对星谱颜色进行加分或扣分。

---

## 关于强化等级

芯核最高可强化到 +15。

每强化 3 级，会随机增加或强化一条副属性。

强化节点为：

- +3
- +6
- +9
- +12
- +15

副属性最多 4 种。  
当副属性达到 4 种后，后续只会强化已有副属性。

从 V0.7 开始，赌核建议会根据当前强化等级调整：

- +0：判断是否值得开赌
- +3：判断是否继续到 +6
- +6：判断是否继续到 +9
- +9：判断是否继续到 +12
- +12：判断是否冲 +15
- +15：只判断成品质量，不再建议继续强化

---

## 注意事项

本工具仅供参考，不是绝对标准。

芯核判断会受到以下因素影响：

- 卡牌练度
- 队伍搭配
- 当前关卡
- 操作手法
- 玩家资源情况
- 是否追求极限伤害
- 是否只是过渡使用

请不要只根据工具结果直接分解芯核。  
分解前请确认芯核没有装备、没有锁定，避免误操作。

## V0.7 更新内容

V0.7 是逻辑修正版，主要修正 V0.6 中容易误导的判断项。

### 主要更新

1. 取消星谱颜色评分  
   星谱颜色不再参与加分或扣分。  
   实战中色谱应根据实际配队需求选择，本工具只判断芯核属性是否适合当前使用身份。

2. 明确“按当前使用身份判断”  
   芯核收益按玩家实战中选择的身份判断，而不是按卡片来源判断。

3. 新增“低优先级 / 过渡词条”档位  
   避免部分并非核心、但也不是废词条的属性被误判为“待确认 +1”。

4. 修复鬼王双暴流下攻击加成评分过低的问题  
   在鬼王双暴流中，攻击加成、虚弱增伤、誓约增伤、攻击现在会被归为低优先级 / 过渡词条。

5. 修正赌核建议与当前强化等级不联动的问题  
   现在工具会根据当前强化等级调整建议：
   - +0 / +3 / +6：给出下一阶段赌核建议
   - +9：判断是否继续到 +12
   - +12：判断是否冲 +15
   - +15：不再输出继续强化建议，只判断保留 / 过渡 / 替换

6. 优化结果说明  
   结果区增加提示：本工具按当前使用身份判断收益，星谱颜色不参与评分。
